# Grammable

Grammable is a simple Instagram clone built using automated software tests (Test-Driven Development) in Ruby. 

## Installation

First, run bundle install:

```bash
bundle install
```

Second, create a database:

```bash
rake db:create
```
```bash
rake db:migrate
```

Third, start your local host:

```bash
rails s
```
